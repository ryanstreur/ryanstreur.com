# ryanstreur.com

I am Ryan Streur, and this is my personal website.

## Motivation

I'm kind of tired of having so many different repositories to hold all my pet projects. When I start a new project to explore some new technology I would like it to be a part of a more cohesive whole. This repo will serve as a trunk for those projects, and will include docker scripts and compose files to orchestrate them to serve that purpose. If I like a project I wrote in the past (statistically unlikely) I'll add it and its history to this repository.

Not sure about deployment target yet. I'll cross that bridge when I come to it.
